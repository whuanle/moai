# MoAI - AI Application Platform

MoAI is a comprehensive AI application platform that provides:

- Multi-model AI assistant support (GPT, Claude, Gemini, etc.)
- Knowledge base management with document embedding and search
- Plugin ecosystem (MCP plugins, OpenAPI plugins)
- Prompt management and optimization
- User authentication with OAuth integration
- Admin panel for system configuration

The platform is designed for both individual users and teams, with role-based access control (admin/user) and collaborative features like wiki sharing.

Primary language: Chinese (Simplified)

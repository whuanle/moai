# MoAI Product Overview

MoAI is an AI-powered platform that provides:

- AI Assistant chat functionality with multiple AI model support (OpenAI, Anthropic, HuggingFace, Mistral, etc.)
- Knowledge base (Wiki) management with document vectorization and semantic search
- Plugin system for extensible AI capabilities (native plugins, custom plugins, tool plugins)
- Prompt template management
- User authentication with OAuth2.0 support (Feishu, DingTalk, WeixinWork)
- Team collaboration features
- File storage (local, S3, MinIO, Aliyun OSS, Tencent COS)
- Workflow automation

The platform consists of a .NET 9 backend API and a React frontend (TypeScript/Vite).

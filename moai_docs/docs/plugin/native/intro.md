# 介绍

内置插件是直接用代码编写的相关功能逻辑，可以实现更加复杂的功能，这些功能直接依赖本地代码实现。

内置插件被用于 AI 应用、流程编排、自定义运行。



![image-20260202183016612](images/image-20260202183016612.png)



在模板页面选择要创建的插件，同一个插件可以创建多个实例。



![image-20260202183052004](images/image-20260202183052004.png)



不同的插件需要的配置不一样，根据要求填写即可。

![image-20260202183147833](images/image-20260202183147833.png)



配置完成后，可以手动运行该插件，你可以以此判断插件是否配置成功。

![image-20260202183258086](images/image-20260202183258086.png)

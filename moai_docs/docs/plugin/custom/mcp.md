# MCP 插件

MCP 插件配置比较简单。



![3fb10141-a51d-4a8d-8e98-7fbaedbc3c92](images/3fb10141-a51d-4a8d-8e98-7fbaedbc3c92.png)



你可以在这里通过 MCP 接入各大平台的服务，以阿里云百炼平台的通义千问-图像生成为例，开通服务后，复制 `baseUrl` 和 Key。

![0ee98d31-a179-4332-bcf1-0665ce76ec59](images/0ee98d31-a179-4332-bcf1-0665ce76ec59.png)

![2b086960-c899-44e0-83c6-5edf64357269](images/2b086960-c899-44e0-83c6-5edf64357269.png)



填写时要注意其通讯方式，百炼平台有些 MCP 服务是 sse，有些是 streamable 格式。

![image-20260202193308250](images/image-20260202193308250.png)

![image-20260202193259410](images/image-20260202193259410.png)



接入 MCP 服务后，你可以在 AI 对话或应用开发中，使用该插件。

![473ed380-2aa3-4417-9dc2-c683410dd0b1](images/473ed380-2aa3-4417-9dc2-c683410dd0b1.png)



你可以在团队中创建应用，并在应用里面添加该插件，那么这个应用就具有对应的能力。



![image-20260202193559212](images/image-20260202193559212.png)
